# Ether Realm Animations

A stunning, immersive DeFi landing page featuring advanced web animations and interactive visual effects.

## Overview

**Ether Realm** is a DeFi (Decentralized Finance) platform landing page with a focus on cutting-edge animations and visual design. The site creates an immersive experience through custom cursor interactions, canvas-based animations, and dynamic visual effects.

## Features

- **Custom Cursor System**: Interactive custom cursor with ring effects that responds to user movement
- **Canvas Animations**: Real-time animated canvas rendering for visual depth
- **Smooth Scroll**: Enhanced scrolling behavior with smooth transitions
- **Responsive Design**: Optimized for multiple screen sizes
- **Modern Typography**: Uses Space Grotesk and Space Mono fonts for a contemporary aesthetic
- **Color Theme**: Rich color palette featuring crystal purples, aurora greens, and plasma effects

## Color Palette

- **Void** (#03020A) - Deep background
- **Deep** (#080618) - Secondary background
- **Crystal** (#6B4FFF) - Primary accent (purple)
- **Aurora** (#00FFB3) - Secondary accent (teal)
- **Plasma** (#FF6BFF) - Tertiary accent (magenta)
- **Gold** (#FFD166) - Highlight accent

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- Google Fonts (Space Grotesk, Space Mono)


2. Open in your browser:
   ```bash
   open animation.html
   ```
